# 🚧 Drive Sense – Road Hazard Detection & Accident Prediction

**Drive Sense** is an AI-powered real-time road hazard detection and accident prediction system using **YOLOv8** and **Computer Vision**. It identifies road threats such as potholes, roadblocks, and sudden obstacles to improve driver safety and support intelligent transport systems.

---

## 🧠 Features

- 🚨 Real-time hazard detection using YOLOv8
- 📹 Dashcam/CCTV video input supported
- 🧠 Accident prediction via object trajectory analysis
- ⚡ High-speed processing via V8 engine integration
- 🧪 Trained on diverse road datasets

---

## 🔧 Tech Stack

- YOLOv8 (Ultralytics)
- Python
- OpenCV
- NumPy, Pandas
- V8 Engine (for optimized execution)
- C++ or JavaScript (if applicable)

---

## 🚀 How to Run

1. Clone the repository  
   `git clone https://github.com/syedsamiullah/drive-sense-ai.git`

2. Install dependencies  
   `pip install -r requirements.txt`

3. Run detection  
   `python yolov8/detect.py --source assets/sample_input.jpg --weights yolov8/model.pt`

> Replace with video source or live cam as needed.
